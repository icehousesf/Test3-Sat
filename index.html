Tracking on the creation of this repository and files:

Following GitHub on command line Part 2 video:

1) from posh~git, create the Test3-Sat folder, run git init

2) from Visual Studio Code (VSC), create the index.html

3) from github.com, create a new repository called "Test3-Sat", it gave the following instructions...

...or create a new repository on the command line

    echo "# Test3-Sat" >> README.md
    git init
    git add README.md
    git commit -m "first commit"
    git remote add origin https://github.com/icehousesf/Test3-Sat.git
    git push -u origin master

...or push an existing repository from the command line

    git remote add origin https://github.com/icehousesf/Test3-Sat.git
    git push -u origin master

4) from posh~git, ran the 2 commands below:

    git remote add origin https://github.com/icehousesf/Test3-Sat.git
    git push -u origin master

5) from posh~git, run the following commands:

    git branch add-more-html (create new branch called "add-more-html")
    git checkout add-more-html

6) from VSC, modify the index.html

7) from posh~git, run the following commands:

    git add index.html (to stage)
    git commit -m '... the commit message ...'
    git push -u origin add-more-html (to push the local branch "add-more-html" to remote)

note: use "git log -n"" to check the commit history

8) from Github.com, click "Compare & pull request" button to create the pull request

9) from posh~git, run the following commands:

    git checkout master
    git merge add-more-html (to merge the add-more-html and the master branches locally)
    git push (to push the "add-more-html" branch to remote

10) from Github.com, delete the "add-more-html" branch
